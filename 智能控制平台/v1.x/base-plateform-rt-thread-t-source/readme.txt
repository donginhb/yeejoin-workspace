==========================================================
m3控制led灯
==========================================================
1号绿色电源灯：通电后常亮
2号黄色led灯：设备运行正常时，2Hz闪烁（现在只有这一种状态）
3号红色led灯：正常时熄灭；非法开锁时，1Hz闪烁
4号红色led灯：正常时熄灭；温度或湿度超过阈值时，1Hz闪烁

温度阈值的默认值是：[3.0, 45.0]
湿度阈值的默认值是：[10.0%, 80.0%]
